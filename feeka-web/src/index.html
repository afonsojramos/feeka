About Us

We are feeka, a Portuguese company that develops technological solutions focused on the client's opportunity with versatility, collaboration, innovation, and creativity. The company, which started its work in 2019, has more than 30 employees and currently has partnerships with companies Sonae BIT, Critical Manufacturing, ZOS Lda and Escola Global.

Projects

Quaerere

Educational application project for students of Escola Global. The school is a reference in investing in innovative technology for teaching and in this project, students will be able to learn with quizzes and tutorials as well as teachers with the possibility of planning the lessons.

Out-of-Stock

A project proposed by Sonae Bit to optimize stock replenishment through a system that detects empty shelves, in order to avoid the bad appearance created by the lack of exposed products. The key point of the project was the use of machine learning as well as image analysis for recognition of the products and their absence.

Electronic Failure Catalogue

The Critical Manufacturing project is based on the MES product, customized for device factories, semiconductors, among others. To improve this type of product, it is necessary that in each step there are errors detections, and so we produce an electronic catalog of errors/failures in which the worker can select by the machine or cell phone and then obtain a list with a report of all the bugs that will be sent to the office later.

ZeroZero

Application that concentrates the streaming information of sporting events. The main purpose of the application is to enable the user to find out which streaming channels are transmitting a particular sporting event. We developed the product in partnership with ZeroZero that provided the necessary API for the execution of the project.
